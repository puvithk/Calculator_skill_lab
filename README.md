# Calculator

This project is a simple calculator built with HTML, CSS, and JavaScript.

## Features

- Basic arithmetic operations: addition, subtraction, multiplication, division, and percentage.
- Clear entry (CE) and clear all (C) functionality.
- Responsive design that works on different screen sizes.

## Usage

Open the `index.html` file in your web browser to use the calculator.

## Design

The calculator has a futuristic cityscape as its background. The main calculator interface is styled with a black background and cyan borders. The input display and buttons have a consistent black and cyan theme. The buttons have hover effects for better user experience.

## Scripts

The calculator's functionality is implemented in the `index.js` script.

## Styles

The styles for the calculator are in the `index.css` file.
